# TUI 快速使用指南

## 1. 启动方式

在项目根目录执行：

```bash
python -m xiaotie.tui.main
```

如果首次启动且未检测到有效配置，会自动进入引导流程。

## 2. Onboarding 引导流程

- 步骤包含：欢迎 -> Provider 选择 -> API Key -> 连接测试 -> 完成
- 支持跳过：`Ctrl+S`
- 每步有进度与错误提示，可回退重试

## 3. 常用快捷键

- `Ctrl+K` 打开命令面板
- `Ctrl+M` 打开模型选择
- `Ctrl+T` 打开主题选择与实时预览
- `Ctrl+B` 切换侧栏
- `Ctrl+N` 新建会话
- `Ctrl+S` 保存会话
- `F1` 打开帮助
- `Ctrl+Q` 退出

## 4. 命令交互

- 输入 `/` 进入命令模式，例如：
  - `/help`
  - `/status`
  - `/themes`
  - `/models`
- 高风险命令会触发确认层：
  - 意图识别
  - 输入 `CONFIRM`
  - 冷却期后执行

## 5. 终端体验建议

- 建议终端宽度 >= 110 列，获得完整双栏体验
- 窄屏下侧栏会自动隐藏，主区仍可完整操作
- 推荐深色背景终端以获得最佳对比度与可读性

## 6. 故障排查

- 若样式异常，先重启终端并重新启动 TUI
- 若配置缺失，删除无效配置后重新触发 Onboarding
- 若命令被拦截，查看高风险确认提示并按规则确认
