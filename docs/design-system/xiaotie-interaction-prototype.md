# 小铁 UI 交互原型（终端版）

## 1. 关键用户流

```mermaid
flowchart LR
  A[启动应用] --> B[输入问题]
  B --> C{普通问题或命令}
  C -->|普通问题| D[Agent 执行]
  D --> E[思考指示 + 状态行处理中]
  E --> F[返回消息 + Toast反馈]
  C -->|命令| G[命令面板/命令执行]
  G --> H[状态更新/会话刷新]
```

## 2. 主界面原型

```text
┌──────────────────────────────────────────────────────────────────────────────┐
│ Header: 小铁 XiaoTie                                                        │
├──────────────────────────────────────────────────────────────────────────────┤
│ 消息主区 (MessageList)                           │ 会话侧栏 (SessionList)   │
```

## 5. 响应式行为原型

- 宽度 >= 130：双栏布局（主区 + 侧栏）。
- 宽度 110~129：侧栏压缩宽度。
- 宽度 < 110：隐藏侧栏，保留主区与输入区。

## 6. 交互反馈原型

- 成功：右上角 `success Toast`。
- 警告：`warning Toast` + 命令提示。
- 错误：`error Toast` + 恢复引导（如 `/help`）。
